# TodoApp

A simple web-based Todo application built for practicing Git and GitHub workflows.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and clean UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository:
   git clone <repo-url>
2. Open the project folder.
3. Open `index.html` in your browser.

## Folder Structure
.
├── index.html
├── style.css
├── script.js
└── README.md
